## Objetivos

- Medir o tempo de execução e comportamento de memória de diversos algoritmos de ordenação e pesquisa.
- Comparar resultados em diferentes entradas (aleatórias, já ordenadas, inversamente ordenadas).
- Gerar saídas para análise e plotagem.
